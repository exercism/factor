* [Björn Lindqvist's Factor Playground](https://github.com/bjourne/playground-factor).
* [Learn X in Y Minutes, where X = Factor](https://learnxinyminutes.com/docs/factor).
* [Factor on RosettaCode](https://rosettacode.org/wiki/Category:Factor).
* The [Concatenative.org Factor Wiki](https://concatenative.org/wiki/view/Factor).
* The [`factor-lang` tag on Stack Overflow](https://stackoverflow.com/questions/tagged/factor-lang).
* The [Factor Mailing List](https://concatenative.org/wiki/view/Factor/Mailing%20list).
* The [Factor IRC Channel](https://concatenative.org/wiki/view/Concatenative%20IRC%20channel).
* The [Joy](https://enwp.org/Joy_%28programming_language%29) and [Forth](https://enwp.org/Forth_%28programming_language%29) programming languages.
* The [Common Lisp Object System](https://enwp.org/Common_Lisp_Object_System).
* [Factor itself](https://github.com/factor/factor).
* [Factor Discord](https://discord.gg/QxJYZx3QDf).
